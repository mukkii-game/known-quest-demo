# KNOWN//QUEST — Game-first M0 prototype

日本人のゲームファンが、ゲーム内ですでに見ていた英語を発見し、一般英語へつなげる体験の初期プロトタイプです。

## Status

- M0の非公開教材を使った公開テスト画面
- 教材・語義・関連・権利はAIドラフト／Human未承認
- 回答はブラウザのメモリ内だけで処理し、保存・送信しない
- API、広告、分析SDK、外部アセットを使用しない
- 固有作品の画像、画面、ロゴ、台詞、競合教材を使用しない

## Feedback guide

試した人には、次の3点を聞いてください。

1. 「思ったより英語を知っていた」と感じたか
2. 説明が多い、分かりにくい、または不自然な語があったか
3. ゲーム英語をもう少し深く続けたいと思ったか

これは完成アプリや公開教材ではありません。内容を正式利用する前にHumanによる内容・権利確認が必要です。
