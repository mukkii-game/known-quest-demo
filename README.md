# KNOWN QUEST — ゲーム英語スキャン

日本人のゲームファンが、ゲーム内ですでに見ていた英語を発見する縦スマホ向けM0テストです。最初に `セガ派 / その他派` を選び、セガ派30作品またはその他派100問へ進みます。

## Status

- その他派100問は初級34・中級33・上級33。セガ派30作品は各10。各ルート内の正規化後の主役英語重複は0件
- その他派はゲームタイトル入口20件。セガ派は日本版の実発売タイトル30作品で、前半15作品をSC/SGからMega Driveまで、後半をSega Saturn以降にした
- セガ派は海外版タイトルへ置き換えず、日本で表示された本番タイトル文字だけを入口にする
- ロゴ、ゲーム画像、画面、台詞、競合教材は不使用。タイトルは文字での参照だけ
- 左は「知らない」、右は「知ってる」。スワイプ、ボタン、マウスドラッグに対応
- 英語はカード登場時に読み上げ、どちらを選んでも短い答えを2秒表示。`知ってる` は短い上昇ジングル、`知らない` は控えめな下降キャンセル音。音は上部ボタンで切替可能
- 2語・3語タイトルは1問で出し、後のマップでは各構成語を元タイトル付きで別々に表示
- 10ブロックの進捗表示と、難度加重した0〜100のゲームスコア。結果はルート別の3段階ファン評価
- 知識マップは一方を選ぶと両方が止まり、主役語と関連語の短い意味を上下2段で比較。その後は画面内のどこをタップしても次へ進む
- 回答は端末メモリ内だけ。保存・送信・分析・広告・外部アセット・APIなし
- 教材、帯、点数、作品名利用、短義、関連語、AI生成仮ガイドはすべてHuman未承認

`GAME ENGLISH` 点数は自己申告を使う遊び上の目安で、語彙力診断、CEFR、正式な能力判定ではありません。内容と作品名利用を正式採用する前に、Humanによる内容・権利確認が必要です。
