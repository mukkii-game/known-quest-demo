# KNOWN QUEST — ゲーム英語100

日本人のゲームファンが、ゲーム内ですでに見ていた英語を発見する縦スマホ向けM0テストです。最初に `セガ派 / その他派` を選び、別々の100問へ進みます。

## Status

- 両ルートとも100問、初級34・中級33・上級33。各ルート内の正規化後の主役英語重複は0件
- その他派はゲームタイトル入口20件。セガ派は日本版の実発売タイトル由来97件と、未検証の `SEGA MANIA` 社内語3件
- セガ派は海外版タイトルへ置き換えず、日本で表示された本番タイトル文字だけを入口にする
- ロゴ、ゲーム画像、画面、台詞、競合教材は不使用。タイトルは文字での参照だけ
- 左は「知らない」、右は「知ってる」。スワイプ、ボタン、マウスドラッグに対応
- 英語はカード登場時に読み上げ、どちらを選んでも短い答えを1秒表示。音声は上部ボタンで切替可能
- `チャンピオンボクシング` は `champion boxing` と2語で出し、後のマップでは `champion` と `boxing` を元タイトル付きで別々に表示
- 10×10の進捗表示と、難度加重した0〜100の `GAME ENGLISH` ゲームスコア
- 知識マップは一方を選ぶと両方が止まり、関係句を表示。その後は画面内のどこをタップしても次へ進む
- 回答は端末メモリ内だけ。保存・送信・分析・広告・外部アセット・APIなし
- 教材、帯、点数、作品名利用、社内語、関係句、AI生成仮ガイドはすべてHuman未承認

`GAME ENGLISH` 点数は自己申告を使う遊び上の目安で、語彙力診断、CEFR、正式な能力判定ではありません。内容と作品名利用を正式採用する前に、Humanによる内容・権利確認が必要です。
