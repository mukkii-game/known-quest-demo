# ゲーマーワードクエスト

おまえはすでに知っている！

ゲームタイトルを入口に、すでに知っていた英語を発掘するスマートフォン／PC向け公開デモです。

## 遊べるモード

- セガ派
- 任天堂派
- ソニー派
- MS／Xbox派

各モードは30タイトルの通常テストと、最大30件の「類語で広げる」で構成します。左は「知らない」、右は「知ってる」。スワイプ、ボタン、マウスドラッグに対応し、回答後は意味を2秒表示して自動で進みます。

語彙パワーと履歴はこのブラウザの `localStorage` にだけ保存します。外部送信、分析、広告、外部アセット、API、追加課金はありません。未出題語は履歴に表示せず、一度覚えた語は後で間違えても取り消しません。

作品名は文字で参照し、ロゴ、ゲーム画像、画面、台詞、競合教材は使用していません。各モード30タイトル・60登録語、合計240語のタイトル選定、意味、難度、関連語、権利状態はAIドラフト／Human未承認です。正式教材、CEFR診断、公式な能力判定ではありません。

## 試してほしい点

1. 知っている作品が続き、英語への自信が少し増えるか
2. 4陣営のタイトル選定に、代表作・年代・少し渋い作品のバランスがあるか
3. 30問、2秒の答え表示、進捗、音が気持ちよいか
4. スマホの縦画面で迷わず片手操作できるか
5. 結果、語彙パワー、履歴によって別陣営も遊びたくなるか
6. 「類語で広げる」でタイトルの別の英単語や意味関係を発見できるか
