# KNOWN QUEST — ゲーム英語100

日本人のゲームファンが、ゲーム内ですでに見ていた英語を100問で発見する縦スマホ向けM0テストです。

## Status

- 100問、初級34・中級33・上級33。正規化後の主役英語重複は0件
- ゲームタイトルを入口にした問題20件。Dragon Questのような見慣れた組み合わせからRedemptionなどへ進む
- ロゴ、ゲーム画像、画面、台詞、競合教材は不使用。タイトルは文字での参照だけ
- 左は「知らない」、右は「知ってる」。スワイプ、ボタン、マウスドラッグに対応
- 10×10の進捗表示と、難度加重した0〜100の `GAME ENGLISH` ゲームスコア
- 知識マップは一方を選ぶと両方が止まり、関係句を表示。その後は画面内のどこをタップしても次へ進む
- 回答は端末メモリ内だけ。保存・送信・分析・広告・外部アセット・APIなし
- 教材、帯、点数、作品名利用、関係句、AI生成仮ガイドはすべてHuman未承認

`GAME ENGLISH` 点数は自己申告を使う遊び上の目安で、語彙力診断、CEFR、正式な能力判定ではありません。内容と作品名利用を正式採用する前に、Humanによる内容・権利確認が必要です。

## Feedback

1. 最初の数問で「知ってる」が続き、自信が出たか
2. タイトル問題は楽しいか。難しいタイトル語へ進む順番は自然か
3. 100問の長さと進捗表示は適切か
4. 点数はもう一度試したくなるか
5. 1択後に関係が出て、どこでもタップで次へ進む操作は直感的か
6. 縦スマホで文字、2択、色、キャラは見やすいか
