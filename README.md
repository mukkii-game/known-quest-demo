# KNOWN//QUEST — Game-first M0 prototype

日本人のゲームファンが、ゲーム内ですでに見ていた英語を発見し、一般英語へつなげる体験の初期プロトタイプです。現在の100問Tinder型画面は操作と候補を試すM0実験で、正式MVPの15語測定を置き換えるものではありません。

## Status

- M0の非公開教材を使った公開テスト画面
- 100問は初級34・中級33・上級33。各主役語に左右2つの関連語を表示
- 左は「知らない」、右は「知ってる」。スワイプ、ボタン、マウスドラッグで操作可能
- 関連語の左右選択には正解・得点がなく、結果は自己申告の既知数。英語力や能力の判定ではない
- 初回の知識ネットワークは「知ってる」と答えた主役語から最大10件
- 100主役語・200関連語・語義・関係句・難度帯・順序・権利はすべてAIドラフト／Human-unapproved
- 回答はブラウザのメモリ内だけで処理し、保存・送信しない
- API、広告、分析SDK、外部アセットを使用しない
- 固有作品の画像、画面、ロゴ、台詞、競合教材を使用しない

## Feedback guide

試した人には、次の3点を聞いてください。

1. 「思ったより英語を知っていた」と感じたか
2. 説明が多い、分かりにくい、または不自然な語があったか
3. 左右スワイプ、ボタン、マウスドラッグのどれが使いやすかったか
4. 100問は長すぎなかったか。初級・中級・上級の切り替わりは自然だったか
5. 各語の左右2関連語と短い関係句で、語どうしのつながりが分かりやすくなったか
6. 最大10件の知識ネットワークを、もう少し深く続けたいと思ったか

これは完成アプリ、公開教材、語彙力診断ではありません。内容を正式利用する前にHumanによる内容・権利確認が必要です。
