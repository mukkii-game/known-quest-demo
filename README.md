# ボキャブラクエスト

お前はすでに知っている！

ゲームタイトルを入口に、すでに知っていた英語を発掘するスマートフォン／PC向け公開デモです。

## 遊べるモード

- セガ派
- 任天堂派
- PS派
- Xbox派
- マイコン族

各モードは30タイトル。左は「知らない」、右は「知ってる」。スワイプ、ボタン、マウスドラッグに対応し、回答後はタイトル内の未学習語すべての意味を2秒表示して自動で進みます。「類語でつなげる」はデモ版では省略しています。

超人強度と履歴はこのブラウザの `localStorage` にだけ保存します。外部送信、分析、広告、外部アセット、API、追加課金はありません。学習済み語は別モードでも最初から意味を表示し、未出題語は履歴に出さず、一度覚えた語は後で間違えても取り消しません。

作品名は文字で参照し、ロゴ、ゲーム画像、画面、台詞、競合教材は使用していません。5モード150タイトル・314学習語レコードのタイトル選定、意味、語形、権利状態はAIドラフト／Human未承認です。正式教材、CEFR診断、公式な能力判定ではありません。

## 試してほしい点

1. 知っている作品が続き、英語への自信が少し増えるか
2. 5ルートのタイトル選定に、代表作・年代・少し渋い作品のバランスがあるか
3. 30問、2秒の答え表示、進捗、音が気持ちよいか
4. スマホの縦画面で迷わず片手操作できるか
5. 結果、超人強度、履歴によって別ルートも遊びたくなるか
6. 一つのタイトルから複数の英単語を発見できるか
